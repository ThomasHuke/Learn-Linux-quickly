# 重定向

[目录](./summary.md)

- ## 有以下命令需要掌握

    - cat － 连接文件

    - sort － 排序文本行

    - uniq － 报道或省略重复行

    - grep － 打印匹配行

    - wc － 打印文件中换行符，字，和字节个数

    - head － 输出文件第一部分

    - tail - 输出文件最后一部分
    
    - tee - 从标准输入读取数据，并同时写到标准输出和文件

- ## 概念
    - 默认的输入流就是键盘。

    - 默认的输出流就是屏幕，重定向就是要更改这个流的走向。

    - 输入流输出流等前缀都是std，例如输出流使用stdout输入stdin错误stderr

    - io重定向到意义：**I/O 重定向允许我们可以更改输出走向和输入来向。一般地，输出送到屏幕，输入来自键盘， 但是通过 I/O 重定向，我们可以改变输入输出方向**
